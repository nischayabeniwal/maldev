# MalDev

A personal malware development and reverse engineering repository created for educational and research purposes only.

## About

This repository contains malware development experiments, proof-of-concepts, reverse engineering exercises, and low-level Windows internals research created and tested by me in a completely isolated lab environment.

The primary goal of this project is to:

- Learn malware development
- Understand offensive security concepts
- Practice reverse engineering
- Study Windows internals
- Explore malware analysis techniques
- Improve red team and exploit development skills

---

## Disclaimer

All projects, samples, and experiments in this repository were:

- Developed strictly for educational purposes
- Tested only inside isolated virtual machines/lab environments
- Never deployed on public systems or networks
- Never intended to harm individuals, organizations, or infrastructure

No unauthorized access, malicious deployment, or harmful activity was performed.

This repository exists solely for learning, research, and defensive security understanding.
